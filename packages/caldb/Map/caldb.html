caldb: Map emulation for the MySQL database

libcaldb.a contains routines that provide a Map-like interface to the CLAS MySQL-based calibration database. It was written to provide backward compatibility for analysis codes that access the Mapmanager. As such, calling arguments are identical to those of the Map routines. At present, the following routines are supported:

  • map_stat_item
  • map_get_char
  • map_get_float
  • map_get_int
  • map_put_char
  • map_put_float
  • map_put_int
See the Mapmanager page for calling arguments.

For more general information on the calibration database, see the CLAS Calibration Database document. You can also browse the contents of the database using the Calibration Database Query Page.
